🏠 Real Estate 3D Analytics Platform

A responsive, cross-device web platform for uploading and visualizing 3D scans of real estate properties with built-in analytics.


📌 About the Project
This is my Experiential Capstone Project at Concordia University Texas. The platform allows users to upload 3D scans of real estate properties and explore them through an interactive, browser-based viewer — accessible on any device, from desktop to mobile.
Key goals:

Make 3D real estate data accessible without specialized software
Surface property analytics through a clean, data-driven UI
Build with a responsive design that works across all screen sizes


✨ Features

📤 Upload and store 3D property scan files
🔍 Interactive visualization of 3D real estate models in-browser
📊 Analytics dashboard surfacing property insights from scan data
📱 Fully responsive — works on desktop, tablet, and mobile
🧭 Clean, user-focused navigation and layout
